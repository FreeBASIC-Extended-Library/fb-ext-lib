This release does not fully support the latest
version of FreeBASIC 0.90, the latest supported
version is 0.24.

On the next screen please provide
the directory where the
FreeBASIC install directory is located.

For example:
If FreeBASIC is installed in C:\FreeBASIC,
you would provide C:\FreeBASIC or
if FreeBASIC is installed in P:\llamas
you would provide P:\llamas

It is recommended you uninstall any previous
versions of the Extended Library before
continuing with the installation.
